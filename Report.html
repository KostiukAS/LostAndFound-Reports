Опис предметного середовища

Часто трапляються випадки, коли люди гублять свої речі, тому вже давно існують місця, які мають назву бюро знахідок, де приймають та зберігають такі речі до моменту повернення їх власнику. Проте, в наш час міста дуже сильно розростаються і з’ясувати до якого бюро віднесли твою загублену річ, та чи віднесли взагалі, стає проблематично. Саме тому ідея створення додатка бюро знахідок, в якому б зберігалась інформація про знайдені речі та за допомогою якого можна було б легко знаходити свої втрачені речі, є дуже актуальною. Саме ця ідея і реалізована в моєму сайті.

HTML-код таблиць

Код таблиці:

Вигляд таблиці:

Теорія:

Тег table визначає HTML таблицю. Кожен рядок таблиці визначається тегом tr. Кожен заголовок таблиці визначається тегом th. Кожна чарунка з даними таблиці визначається тегом td. За замовчуванням текст в елементах th виділений жирним шрифтом і відцентрований. За замовчуванням текст в елементах td є звичайним і вирівняним за лівим краєм. Щоб додати межу до таблиці, використовуйте CSS властивість border. Щоб дозволити межам таблиці згортатися в одну межу, додайте CSS властивість border-collapse. Щоб встановити внутрішній відступ, використовуйте CSS властивість padding. Щоб зробити чарунку, що охоплює більше одного стовпця, використовуйте атрибут colspan.

HTML-код зображення

Код зображення:

Вигляд зображення:

Теорія:

В HTML зображення визначаються за допомогою тега img. Тег img пустий, містить лише атрибути і не має закриваючого тега. Атрибут src вказує URL (веб-адресу) зображення. Значення атрибута alt має описувати зображення, тобто те, що на ньому зображено. Атрибути width и height завжди визначають ширину і висоту зображення в пікселях. Використовуйте CSS властивість float, що дозволяє зображенню плавати праворуч або ліворуч від тексту.

HTML-код списків

Код списків:

Вигляд списків:

Теорія:

Невпорядкований (ненумерований) список починається з тега ul. Кожний елемент списку починається з тега li. CSS властивість list-style-type використовується для визначення стиля маркера елемента списку. Впорядкований (нумерований) список починається з тега ol. Кожний елемент списку починається з тега li. Атрибут type тега ol визначає тип маркера елемента списка. Тег dl визначає список опису, тег dt визначає термін (ім’я), а тег dd описує кожний термін.

HTML-код форми

Код форми:

Вигляд форми:

Теорія:

HTML елемент form визначає форму, яка використовується для збору користувацького вводу. Елемент input є найбільш важливим елементом форми. Атрибут action визначає дію, яку буде виконано при відправці форми. Атрибут target вказує, чи буде відправлений результат відкриватися в новій вкладці браузера, у фреймі або в поточному вікні. Атрибут method вказує метод HTTP (GET або POST), який буде використовуватись при відправці даних форми. Елемент fieldset використовується для групування пов’язаних даних у формі. Елемент legend визначає заголовок для елемента fieldset.
